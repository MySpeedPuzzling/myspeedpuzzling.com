# Illustration Prompts

Reusable ChatGPT image generation prompts for MySpeedPuzzling illustrations.

## Chosen Style: Semi-flat Outlined

See [style-prefix.md](style-prefix.md) for the full style prefix that must be included in every image generation prompt.

**Key traits**: Dark navy outlines (~2px), soft pastel fills with subtle gradients, rounded geometry, speed/motion dashes, floating decorative dots, warm and approachable mood.

## Prompt Files

| File | Purpose |
|------|---------|
| [style-prefix.md](style-prefix.md) | The reusable style prefix — copy into every prompt |
| [subjects/](subjects/) | Individual subject prompts (combine with style prefix) |
