## How Google ranks — and where we now stand (context for priorities)

Google's documented systems reward, in rough order of leverage for us: (1) **links/authority** (PageRank — still the gap vs speedpuzzling.com's exact-match domain; items 2–3 above), (2) **relevance + content quality** (helpful-content signals — homepage restructure, guides with unique measured data, hub pages), (3) **crawlability/indexability** (fixed: sitemaps, hreflang, canonicals, internal links, soft-404s), (4) **page experience** (CWV — improved; Cloudflare would finish it), (5) **engagement/brand signals** (your brand queries already dwarf competitors'; WebSite schema + crossroads root now feed the site-name system). On-page is now near-ceiling; the remaining distance to #1 is authority + content velocity, which are items 2, 3 and 10.

## GSC indexing review (2026-07-11, evening)

Investigated the two alarming GSC signals after the sitemap resubmission. Verdicts:

15. **"Nelze načíst" on 22 of 31 child sitemaps → NOT a real failure.** Production Traefik access logs show Googlebot fetched **every** child sitemap at 08:38 UTC with HTTP 200 in 125–214 ms (puzzles-2..21 even twice). All files are within protocol limits (largest: players 39,762 URLs / 4.6 MB, images-1 20,000 entries / 5.5 MB, cap is 50,000 / 50 MB). The failing rows all have an empty "last read" date — this is the well-known GSC quirk where freshly discovered child sitemaps display "Couldn't fetch" until processing completes. **Action: none. Re-check the report in 3–7 days; only investigate if rows still show no last-read date then.** No size-lowering or performance work needed — response generation is ~150 ms.
16. **1.59M unindexed pages → mostly self-inflicted URL noise, not a penalty.** Breakdown: 896k "blocked by robots.txt" (per-entity action links: stopwatch, add-time, suggest-change, favorites, and the topbar feedback link that embeds each page's own URL as `?url=` — one unique blocked URL per page per locale); 423k "crawled – currently not indexed" (real puzzle/player pages across 6 locales — the authority/thin-content equilibrium, improves with links + slugs #138, not a technical bug); 168k "alternate with proper canonical" (healthy); 67k 403-blocked (old bot-blocking episode, already validated as fixed); 10k redirects + 9k 404s (expected after URL migrations); 5xx count is zero. Fixed today: `rel="nofollow"` on the topbar/export feedback links, and robots.txt gaps — the overhaul robots dropped per-entity action routes the old file (or Google's treatment) had covered: `puzzle_add` (all 6 locale paths), `puzzle_suggest_change` (wildcard rules), `stopwatch_puzzle` de/fr (`/de/puzzle-stoppuhr`, `/fr/puzzle-chronometre`), `compare_players.en` (mounted at domain root, no `/en` prefix), `buy_membership`, collection add/move/remove, borrow/lend settings. Watch the "blocked by robots.txt" count level off, then slowly decay (Google drops blocked URLs from the report lazily — months, not weeks).
